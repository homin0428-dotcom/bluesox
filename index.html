Firebase 연결 설정
블루삭스 팀 실시간 데이터 공유
팀 전체가 같은 데이터를 실시간으로 공유하려면 Firebase 설정이 필요합니다.
① Firebase 프로젝트 생성

console.firebase.google.com → 프로젝트 추가 → 이름 입력 → 생성

② Firestore 데이터베이스 활성화

좌측 메뉴 → Firestore Database → 데이터베이스 만들기 → 테스트 모드 → asia-northeast3(서울) → 완료

③ 웹 앱 등록 후 설정 코드 붙여넣기

프로젝트 설정(⚙️) → 내 앱 → 웹 앱 추가(</>) → firebaseConfig 객체 전체 복사

💡 설정 후 이 HTML 파일을 Netlify에 배포하면 팀원 모두 같은 URL로 실시간 공유됩니다.
